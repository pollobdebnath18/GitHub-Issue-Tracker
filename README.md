1️⃣ Difference between var, let, and const

=> 1. var is old version . let & const is modern version. 2. var not maintain block scope . let & const maintain block scope . 3. var , let & const maintain function scope. 4. var ---> redeclare with same name by var . let & const ---> Can't redeclare with same name by let and const . 5. var maintain function scope not maintain block scope. but let and const maintain all scope.



2️⃣ What is the spread operator (...)

=> The spread operator (...) is used to expand or copy elements from arrays or objects. It allows you to take values from one array or object and place them into another. This is very useful when you want to copy arrays, merge arrays, or clone objects without modifying the original data.



3️⃣ Difference between map(), filter(), and forEach()

=>map() is used to change every item in an array and create a new array.
filter() is used to select some items from an array based on a condition.
forEach() is used to loop through an array and perform an action, but it does not return a new array.



4️⃣ What is an arrow function

=>An arrow function is a shorter and cleaner way to write functions in modern JavaScript.it uses the => symbol instead of the function keyword. Arrow functions make the code more readable and are commonly used when working with array methods like map() or filter().




5️⃣ What are template literals
=> template literals are a way to write strings using backticks ()** in JavaScript.they allow you to insert variables inside a string using ${}`, which makes writing dynamic text easier.
